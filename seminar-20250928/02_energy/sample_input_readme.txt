# 省エネ計算（モデル建物法）サンプル入力

## ファイル説明
このフォルダには省エネ計算のサンプル入力ファイルが含まれています。

### sample_input.csv
モデル建物法計算に必要な最小限の入力項目のサンプルです。

## 入力項目の説明

### 建物基本情報
- 建物名: 物件の識別名
- 用途: 事務所、共同住宅、店舗等
- 階数: 地上階数
- 延床面積: 総床面積（㎡）
- 構造: RC造、S造、木造等

### 外皮性能
- 外壁面積: 外壁の総面積（㎡）
- 窓面積: 開口部の総面積（㎡）
- 屋根面積: 屋根の面積（㎡）
- UA値: 外皮平均熱貫流率（W/㎡K）
- ηAC値: 冷房期平均日射熱取得率

### 設備仕様
- 空調方式: ビル用マルチ、ルームエアコン等
- 空調効率: APF、COP等
- 照明器具: LED、蛍光灯等
- 給湯方式: 電気、ガス等

## 使用方法

1. sample_input.csvをExcelで開く
2. 実際の建物データに書き換え
3. 計算ツールに入力
4. BEI値を確認

## 注意事項

- 数値は適合性判定機関の様式に合わせてください
- 不明な項目は設計者に確認してください
- 計算結果は必ず専門家に確認してもらってください

## セミナーでの使用

セミナーでは以下の流れで実演します：

1. サンプルデータの確認（2分）
2. 計算ツールへの入力（5分）
3. BEI値の算出と評価（3分）

※ 実際の物件では詳細な検討が必要です
